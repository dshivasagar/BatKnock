# General Sans Font Files

Place the following font files in this folder:

  GeneralSans-Regular.otf
  GeneralSans-Medium.otf
  GeneralSans-Semibold.otf
  GeneralSans-Bold.otf

Download General Sans free from:
  https://www.fontshare.com/fonts/general-sans

Steps:
1. Download from fontshare.com
2. Open the zip → "Fonts" → "OTF" folder
3. Copy the 4 files listed above into this folder (assets/fonts/)
4. Run: npx expo start --clear --lan

The app will automatically use General Sans once the files are present.
If the files are missing, the app falls back to the system font (SF Pro on
iOS, Roboto on Android) with no errors.
